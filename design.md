# 📦 Media Library Manager (가칭)

개인 보유 미디어 파일 관리용 데스크탑 프로그램

> **목표**
> 내가 어떤 파일을 가지고 있는지만 확인하기 위한 프로그램
> (재생 목적 아님)

---

## 🎯 프로젝트 목적

- 보유 중인 영상 파일 목록 관리
- 여러 외장하드에 흩어진 파일 통합 관리
- 외장하드가 연결되어 있지 않아도 목록 유지
- 개인 사용 목적 Desktop Application

본 프로젝트는 **미디어 플레이어 / 스트리밍 서버가 아님**

---

## ✅ 현재 개발 범위

### 핵심 기능

- 특정 폴더 수동 스캔
- 파일명 수집
- 품번(Code) 자동 추출
- 작품 자동 생성
- 분할 파일 하나의 작품으로 묶기
- 탐색기에서 파일 위치 열기

---

### 보기 모드

- 썸네일(갤러리) 모드
- 테이블 모드

---

## ❌ 현재 제외 기능

- 영상 재생 기능
- 온라인 메타데이터 자동 수집
- 파일 자동 삭제 / 이동
- 서버 기능
- 로그인 시스템

---

## 🧠 작품 인식 규칙

작품 기준은 **품번(Code)**

예시:

```
AAA65-CD1.mp4
AAA065.mp4
AAA-65(1).avi
```

모두 다음으로 정규화됨:

```
AAA-65
```

→ 하나의 작품으로 관리

---

## 🗂 데이터 구조 개념

주요 엔티티:

- 작품 (Work)
- 파일 (File)
- 드라이브 (Drive)
- 배우 (Actor, 향후 확장 예정)
  ※ 메타데이터 구조만 생성하고 현재는 사용하지 않음

## 🛠 기술 스택

- Language: Python
- Database: SQLite
- UI: PySide6 (예정)
- Editor: VSCode
- Target: Windows Desktop (.exe)
