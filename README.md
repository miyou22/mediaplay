# Media Library Manager

개인용 영상 라이브러리 관리 프로그램.

외장하드 및 로컬 저장소에 흩어진 영상 파일을
정리, 검색, 확인하기 위해 제작한다.

배포 목적보다는 개인 사용을 우선으로 하지만,
구조를 정리하여 추후 공개 배포 가능성을 고려한다.

---

## 🎯 목표

- 여러 디스크에 분산된 영상 관리
- 실제 파일 존재 여부 확인
- 중복 파일 확인
- 메타데이터 기반 검색
- 배우 데이터베이스 구축

---

## 🧩 주요 기능 (예정)

- 특정 폴더 수동 스캔
- 파일 목록 DB 저장
- 썸네일 관리
- 메타데이터 확장 구조
- 배우 정보 관리
- 한국어 / 일본어 메타데이터 지원

---
